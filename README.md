# SuperStore-Sales-DataSet
* Analyzed and visualized two years of Superstore sales data to identify business trends and performance insights.

* Cleaned and transformed raw datasets using Power Query Editor for accurate and analysis-ready reporting.

* Built a star schema data model in Power BI for sales, customer, product, and regional analysis. 

* Developed interactive Power BI dashboards to analyze sales performance, customer segments,and profit trends. 

* Created DAX measures and KPI-driven visualizations to track key metrics and business performance. 

* Implemented sales forecasting and dynamic filters to generate actionable insights for decision-making.
